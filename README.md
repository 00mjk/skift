# About skift**OS**
skift**OS** is a handmade operating system build from scratch.

# Featuring
- Multitasking
- File system
- Userspace

# Build-it
```sh
# Build the toolchain
./TOOLS init

# Build the operating system
./TOOLS buildtarget all

# Run it in qemu
./TOOLS run qemu
```
